# Techniker Berechnungstools

Sammlung von Berechnungs- und Nachschlagetools für die Prozessmesstechnik — entwickelt für Servicetechniker im Bereich Durchfluss, Füllstand, Druck, Temperatur und Analytik. Gehostet auf GitHub Pages, läuft vollständig im Browser ohne Server.

> **Hinweis:** Dieses Projekt ist kein offizielles Produkt von Endress+Hauser. Produktnamen wie Promag®, Promass®, Prowirl®, Cerabar®, Micropilot® und Levelflex® sind eingetragene Marken der Endress+Hauser Gruppe. Die genannten Marken werden ausschließlich zur informativen Referenz verwendet.

## Features

- **Design** — GitHub-inspiriertes Design mit Dark/Light Mode und Kategorie-Farbkodierung
- **Mobil** — Optimiert für Smartphone-Nutzung (kollabierbare Info-Box, gestapelte Tabellen, Touch-optimiert)
- **Live-Berechnung** — Ergebnisse aktualisieren sich bei jeder Eingabe
- **Verwandte Tools** — Kontextuelle Empfehlungen am Ende jeder Seite
- **Tastenkürzel** — `Ctrl+C` (Kopieren), `Esc` (Zurücksetzen), `Ctrl+P` (PDF), `Ctrl+L` (Teilen)
- **Teilen** — URL-Parameter für vorausgefüllte Werte
- **PDF-Export** — Auf allen Tools verfügbar
- **Numerische Tastatur** — Automatisch auf Mobilgeräten bei Zahleneingaben
- **Kategoriefilter & Suche** — Schnellzugriff auf der Startseite
- **Favoriten** — Persönliche Favoritenliste mit Drag & Drop
